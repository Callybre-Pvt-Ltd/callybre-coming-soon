# Callybre Website

A modern, visually appealing "Coming Soon" landing page for Callybre. 

## Overview
Built with React and Vite, this project features a responsive design that adapts its aesthetic based on screen size:
- **Medium & Large Screens:** Features an interactive `LightRays` background effect and rich gradients.
- **Mobile Devices:** Uses a clean, solid background (`#06081a`) to ensure performance and readability on smaller screens.

## Features
- Dynamic text animations using the `SplitText` component.
- Responsive design tailored for varying device sizes.
- Styled with Tailwind CSS.

## Getting Started
To run the project locally:
1. Clone the repository.
2. Run `npm install`, `yarn install`, or `pnpm install`.
3. Start the development server with `npm run dev`, `yarn dev`, or `pnpm dev`.
